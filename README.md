## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Rocket.png" alt="Rocket" width="25" height="25" /> The Mission

**The Ascendant Protocol** is my personal roadmap and progress tracker for my upcoming career switch to a **Django Backend Developer**. 

After finding that generic roadmaps online lacked the depth and specific rigor I needed, I built this offline application to centralize my preparation. You might see my name, **Subash**, throughout the app—this is my personal command center for the "Django Protocol."

### ⚡ Execution
Simply run the application using the shortcut provided in the root directory:
1.  Open the home folder.
2.  Double-click **`shortcut.exe`** (or `Ascendant.exe.lnk`) to launch the protocol.
